## Architecture

MF-MSKA-Net combines three complementary modules inside a compact encoder-decoder:

```
Input MRI (256×256×1)
    ├── CNN Encoder  (3-level, 32/32/64 channels)
    ├── MSFE  ─── Multi-Scale Frequency Encoder
    │             (spatial dilated CNN + FFT magnitude, SE-gated fusion)
    ├── ViT Encoder (4 layers, 6 heads, d=192, patch=16×16)
    ├── SKA   ─── Morphological Skeleton Attention
    │             (coarse mask → skeleton proximity map)
    └── HCAF Decoder (2 cross-attention levels + 2 skip levels)
               └── 1×1 Conv + Sigmoid → Tumour Mask
```

| Module | Role | Params |
|--------|------|--------|
| MSFE | Multi-scale spatial + frequency encoder | ~0.31 M |
| SKA | Skeleton-guided structural attention | ~0.02 M |
| HCAF | Local-global cross-attention fusion | ~0.61 M |
| CNN/ViT backbone | Hierarchical feature extraction | ~0.59 M |
| **Total** | | **1.53 M** |

## Quick Start

### 1 — Clone the repository

```bash
git clone https://github.com/oussama123-ai/MF_MSKA_Net.git
cd MF_MSKA_Net
```

### 2 — Install dependencies

```bash
# Create and activate a virtual environment (recommended)
python -m venv venv
source venv/bin/activate        # Linux / macOS
# venv\Scripts\activate         # Windows

pip install --upgrade pip
pip install -r requirements.txt
```

> **GPU users:** Install the CUDA-enabled TensorFlow build:
> ```bash
> pip install tensorflow[and-cuda]>=2.15.0
> ```

### 3 — Download the BRISC 2025 dataset

The dataset is publicly available on Kaggle:
<https://www.kaggle.com/datasets/briscdataset/brisc2025>

```bash
# Option A: Kaggle CLI
pip install kaggle
kaggle datasets download -d briscdataset/brisc2025
unzip brisc2025.zip -d ./data

# Option B: opendatasets (interactive, asks for Kaggle credentials)
python -c "
import opendatasets as od
od.download('https://www.kaggle.com/datasets/briscdataset/brisc2025',
             data_dir='./data')
"
```

Expected layout after download:
```
data/
└── brisc2025/
    └── segmentation_task/
        └── train/
            ├── images/
            └── masks/
```

### 4 — Train the model

```bash
python scripts/train.py \
    --data_dir  ./data/brisc2025/segmentation_task/train \
    --output_dir ./runs/exp1 \
    --epochs 50 \
    --batch_size 8 \
    --lr 1e-4
```

All hyperparameters match those reported in the paper (Table 5).
The best checkpoint is saved to `./runs/exp1/mf_mska_best.keras`.

### 5 — Evaluate (test set + optional CV + optional ablation)

```bash
# Test set only
python scripts/evaluate.py \
    --data_dir   ./data/brisc2025/segmentation_task/train \
    --ckpt_path  ./runs/exp1/mf_mska_best.keras \
    --output_dir ./runs/exp1

# Test set + 5-fold CV
python scripts/evaluate.py \
    --data_dir   ./data/brisc2025/segmentation_task/train \
    --ckpt_path  ./runs/exp1/mf_mska_best.keras \
    --output_dir ./runs/exp1 \
    --run_cv

# Full evaluation (test + CV + ablation — ~4× training time)
python scripts/evaluate.py \
    --data_dir   ./data/brisc2025/segmentation_task/train \
    --ckpt_path  ./runs/exp1/mf_mska_best.keras \
    --output_dir ./runs/exp1 \
    --run_cv --run_ablation
```

Results are saved to `./runs/exp1/evaluation_results.json`.

### 6 — Predict on new images

```bash
# Single image
python scripts/predict.py \
    --input  ./my_mri_slice.png \
    --ckpt   ./runs/exp1/mf_mska_best.keras \
    --output ./predictions \
    --tta

# Batch of images
python scripts/predict.py \
    --input  ./my_mri_folder/ \
    --ckpt   ./runs/exp1/mf_mska_best.keras \
    --output ./predictions
```

### 7 — Interactive demo

```bash
# Local demo
python scripts/demo.py --ckpt ./runs/exp1/mf_mska_best.keras

# Shared public URL (temporary Gradio link)
python scripts/demo.py --ckpt ./runs/exp1/mf_mska_best.keras --share
```

## Reproducibility

The exact 70/10/20 split is fully reproducible with `seed = 42`.
Split indices are saved automatically to `split_indices.json` during training.

To verify:

```python
from src.data import build_dataframe, split_dataset

df = build_dataframe("./data/.../images", "./data/.../masks")
train_df, val_df, test_df = split_dataset(df, seed=42)
# train=2752, val=394, test=787
```
